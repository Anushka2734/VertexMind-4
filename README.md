# Task Manager Application

A full-stack Task Management application built using the MERN stack.  
This project includes user authentication, JWT-based authorization, and complete CRUD operations for tasks.

## 🚀 Features

- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Create Tasks
- View Tasks
- Update Tasks
- Delete Tasks
- Task Completion Status
- Responsive UI
- REST API Backend

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- React Router DOM
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt
